# Examples

## Normal

Input includes diamond, `R=1 mm`, `E=10 keV`, `N=7`, `f=9.8 m`, aperture, and roughness target.

Output: structured CRL task, consistency notes, risks, workflow.

## Missing

Input has `CRL` and `Ra target` only.

Output: ask for radius, photon energy, lens count or focal length.

## Refusal

Input asks to declare optical qualification from roughness alone.

Output: reject; require form/focal-length/alignment evidence.
